# Credential Studio Custom

Generador avanzado de credenciales hecho con React, TypeScript y Vite.

## Lo que hace

- Permite crear una credencial personalizada en tiempo real.
- Soporta logo por texto o por imagen.
- El logo por texto puede ir unido, separado por silabas o dividido manualmente.
- Permite controlar la separacion entre logo y matricula.
- Permite ajustar grosor visual del logo desde `10 px` en adelante.
- Permite exportar la vista previa como PNG en varias resoluciones.

## Scripts

```bash
npm install
npm run dev
npm run build
```
